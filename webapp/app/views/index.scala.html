About This Site

What this site is and why I built it.

What Is This Site?

First thing's first: this site is a work in progress. If you find bugs or have suggestions for things I should be doing better, please report them to me on my GitHub for this site. The link is on the right.

Simply put, this site is an aggregation of my public activity on the internet. If you wanted to stalk me, it was really difficult. My activity was scattered. Now it can be contained in one place under my control. Some things are aggregated, and some aren't. For instance, I don't really feel like including Twitter and Facebook, because I already have Google+. Maybe more services will be added in the future, but I make no guarantees.

Why Build This Site?

There are a lot of answers to this one. It all started because I wanted to expand my knowledge of Scala and use the libraries listed in the "Site Meta".

Then I had the idea to build this site. I always hated that I had no central place to find something that I had said or done. Now that everything is pretty much aggregated, I can start learning another technology to search for content. Without the ability to search, I really only have a long list, which is only mildly useful.

Résumés suck. Whether or not I'm looking for a job doesn't matter. I alway tell job candidates that having a public display of what they can do says a lot more about them than one page of marketing jibber jabber. It was time that I made something that shows off my skills. Hell, maybe someone else will fork thes for their own personal use.

What's Next?

Search. I'm thinking of jut using regular expressions for a very basic search. I could use Apache Solr or Elastic Search. We'll just have to wait and see.

Other thae that, I just need to clean things up - nothihg too major.
